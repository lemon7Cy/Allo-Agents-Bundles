# 六维评价指标库（判分锚点 + 深读硬扣分项）

评分用 **0–100**，与 `render_eval.py` 的 schema 一致。打分前**先读完本文件**，对每一维按下面的“分档锚点”落档，再写一句**引用草稿具体内容**的依据（不得空泛）。锚点取自真实课程报告样本，覆盖弱/中/强三档。

> 共识口径：本指标库在出题（定标准）、学生端（自评）、评审端（打分）**保持同一把尺子**（各 bundle 各放一份，内容一致）。

---

## 维度一 · 创新性
| 档 | 分段 | 锚点 |
|---|---|---|
| 强 | 90–100 | 提出自己的方法/视角/本地化策略，非套模板；有可复用的新机制。 |
| 中 | 55–75 | 成熟方法的合理应用 + 少量加分项（如双模型对比、跨域类比），真正的原创只停留在“展望”。**(样本典型档)** |
| 弱 | 0–35 | 完全套模板/高度复刻某篇范文的思路，无任何自己的东西。 |

## 维度二 · 数据分析深度
| 档 | 分段 | 锚点 |
|---|---|---|
| 强 | 85–100 | 真实数据 + 多方法对比 + 调参 + 多指标互补 + 残差/bias-variance 分析。 |
| 中 | 55–70 | 有对比/抗噪/消融实验 + 多指标，但数据多为仿真、以表格为主、缺曲线/统计深挖。 |
| 弱 | 0–40 | **关键实验用模拟数据冒充真实**、真值不真实（如 SOC 真值恒定）、只给数值不深挖误差来源。 |

## 维度三 · 完整性
| 档 | 分段 | 锚点 |
|---|---|---|
| 强 | 85–100 | 问题→建模→（手算实例/代码）→实验→可视化→局限展望，端到端闭环。 |
| 中 | 65–80 | 章节框架完整、工程闭环在，但**关键节正文空缺**（如“结果分析”只有小标题无文字）、虎头蛇尾。 |
| 弱 | 0–45 | 残缺，缺多个核心环节（无方法/无结果/无结论）。 |

## 维度四 · 文献引用
| 档 | 分段 | 锚点 |
|---|---|---|
| 强 | 85–100 | 文献规范（DOI/ISBN）+ **正文角标** + 引文与列表**闭环**。 |
| 中 | 50–70 | 文献表本身规范，但**无正文角标**、存在“列了没引 / 引了没列”的对应缺口。 |
| 弱 | 0–30 | **仅 1–2 篇且正文无引用标注**，与综述内容不匹配；或**零参考文献**。 |

## 维度五 · 结论合理性
| 档 | 分段 | 锚点 |
|---|---|---|
| 强 | 85–100 | 结论由证据严格支撑 + 主动标注适用边界/局限。 |
| 中 | 60–80 | 结论与结果一致、诚实列局限；小瑕疵：玩具样例结论略作外推，或**相对提升掩盖绝对误差偏高**（如 SOH RMSE 12% 却称“高精度”）。 |
| 弱 | 0–45 | 结论建立在模拟数据上证据不足、**与图表数据自相矛盾却未讨论**。 |

## 维度六 · 格式规范性
| 档 | 分段 | 锚点 |
|---|---|---|
| 强 | 85–100 | 章节/编号/公式/图表题注/参考文献全规范。 |
| 中 | 65–80 | 学术排版规范，小扣分：图号断档（如缺“图 5.2”）、中英混排、个别错字/未闭合括号。 |
| 弱 | 0–45 | 公式渲染错乱/下标丢失、图表无编号题注、无参考文献、正文留空。 |

---

## 评分落点规则（防评分者发散）
- **分档之间的空隙**（如某维弱档上界与中档下界之间无锚点）：按与两侧锚点的贴近程度**线性内插**落分，不许跳档、不许因"介于两档"而随意取整。
- **纯文本投稿的图表口径**：md/txt 投稿中引用"图X/表X"但文件本身无法嵌入图片时，核**编号连续性与文图引用一致性**即可，**不因"无实际嵌入图"判硬伤**（那是载体限制，不是学生错误）。

## 深读硬扣分项（跨维一致性核验 —— 评价 agent 必须逐项过）
表层关键词/字数/查重抓不到这些，**必须跨模态深读（公式/图/代码/表）后核验**。命中任一条，在对应维度落到“弱”档并在依据里点名：

1. **数据真实性 ↔ 结论一致性**：作者自认（或可证）用模拟/合成数据，却下“验证有效/高精度”结论 → 重扣“数据分析深度 + 结论合理性”。
2. **引用闭环**：参考文献“列了没在正文引”/“正文引了没列入文献表” → 扣“文献引用”。
3. **图表编号连续性**：正文图号/表号断档或与实际嵌入数不符 → 扣“格式规范性”。
4. **关键数值跨章节自洽**：手算/递推/表格数值前后对不上（如协方差递推与汇总表矛盾）→ 扣“结论合理性 + 格式规范性”。
5. **相对 vs 绝对指标**：用“相对提升 X%”掩盖“绝对误差仍很高” → 扣“结论合理性”。
6. **评价基准正确性**：在已知真值的仿真里却用“含噪观测”而非“真值”做 RMSE 基准（误差越小只是越贴噪声）→ 扣“数据分析深度”。

---

## 难度分层（同一把尺子，门槛不同）
- **进阶/联合作品**（多状态、跨时间尺度耦合、多算法集成、含消融实验）：在“完整性、方法集成度、消融充分性”上**提高门槛**（同样的完整度，进阶作品要做到更多才算强档）。
- **基础作品**（单一方法、短篇）：按上表正常落档。
- 难度判断写进评语，避免“进阶作品因为内容多就自动高分”。
